Developer Info

Welcome to the developer details page for the Bug Resolver AI project. This project is part of the AI GitLab Hackathon.

Project: Bug Resolver AI Agent
Developer: Anmol Shukla
Hackathon: AI GitLab Hackathon
Technology stack: Flask, SQLAlchemy, LocalStorage-based frontend settings + AI code-fix generator modules.
Key features: issue parsing, automatic fix generation, test generation, multi-tab settings panel, theme/persistence settings, profile updates.

How to use settings

  1. Open Settings.
  2. Switch tab and update toggles/values.
  3. Click Save Changes - data is persisted in localStorage.
  4. Refresh page - your active tab and settings remain active.
  5. Navigate to Profile to verify display name and email updates.
